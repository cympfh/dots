# Todo スキル

Claude Code のための TODO 管理スキルです。`TODO.md` を読み取り、タスクを自律的に実行します。

---

## /todo: TODO タスクの実行

1. `TODO.md` を読み込む
2. 未完了タスク（`[ ]`）を確認する
3. 優先度・依存関係を考慮して実行順序を決定する
    - 基本は上にあるほど優先とする
4. タスクをちょうど一つ選ぶ
5. 選んだタスクを実行する
6. 実装が完了したら
    - `Bash(date "+%Y-%m-%d %H:%M")` を使用して、正確な完了日時時刻を確認
    - `TODO.md` を更新: チェックマークを入れ、完了日付時刻を記入する
7. タスクを一つ完了した旨をユーザーに報告する

---

## /todo add <prompt>: タスクの追加

1. ユーザーからのプロンプトを受け取る
    - 省略された場合は、ユーザーにタスクの内容を尋ねる
2. 受け取った内容をタスクとして `TODO.md` に追加する
    - タスクは `## [ ] タスク名` の形式で追加する
    - タスクの説明をタスク名の下に記入する
    - 必ず TODO.md の末尾に追加すること

---

## /todo auto-add: タスクの自動追加

1. 実装・実行するべきタスクを自動的に分割する
2. 一つずつ優先度順に `TODO.md` に追加する
    - タスクの内容や優先度は、実装の内容や依存関係を考慮して決定する
    - タスクは `## [ ] タスク名` の形式で追加する
    - タスクの説明をタスク名の下に記入する
    - 必ず TODO.md の末尾に追加すること

---

## /todo list: タスクの一覧表示

1. `TODO.md` を読み込む
2. タスクの一覧をユーザーに表示する
    - タスクの状態（完了/未完了）を明示する
    - タスクの内容や説明も表示する

---

## TODO.md のフォーマット

```markdown
@CLAUDE TODO List

## [x] データベースの設計と実装 [2026-01-01 12:32 完了]
データベースはAWS RDSを使用して、ユーザーデータとタスクデータを管理する。

## [ ] ユーザー認証の実装
JWT を使った認証を実装する。
```

- `[ ]` : 未完了タスク
- `[x]` : 完了タスク（完了日付を `[YYYY-MM-DD HH:MM 完了]` 形式で記入）
- タスクの詳細や変更内容はタスクの説明として記録する

## タスク実行のルール

- **基本は上から順に実行**するが、以下の場合は順番を変えることができる
  - 優先度が明示されている場合
  - 実装の依存関係がある場合（先に完了すべきタスクがある場合）
- タスク完了後は必ず `TODO.md` を更新する
  - `[ ]` → `[x]` に変更
  - `[YYYY-MM-DD HH:MM 完了]` を見出しに追記する
  - TODO.md に記載された内容から変更があった場合は、変更内容を説明に追記する
- ユーザーから新しい要望があれば、`TODO.md` に新しいタスクを追加することも許される

---

## `/todo init`: TODO.md の初期化

`TODO.md` が存在しない場合は、`/todo init` で初期化する。
既に `TODO.md` が存在する場合は上書きせず、既存ファイルがある旨を伝える

以下のテンプレートで `TODO.md` を作成する：

```markdown
@CLAUDE TODO List

## [ ] タスク名
タスクの説明をここに書く。
```

その後の推奨アクション

- .gitignore に TODO.md を追加する
    - ユーザーに許可を貰った上で行うこと
- タスクを追加する
    - auto-add の使用を提案する

---

## まとめ

- `/todo init` : TODO.md を新規作成して初期化する（ファイルが無い場合）
- `/todo auto-add` : すべきタスクを分割し、優先度順に TODO.md に追加する
- `/todo add [prompt]` : 指定した内容をタスクとして TODO.md に追加する
- `/todo list` : TODO.md のタスク一覧を表示する
- `/todo` : TODO.md を読んでタスクを一つ実行する
- 「TODO を確認して」: TODO.md の現在の状態を確認・報告する
- 「〇〇を TODO に追加して」: 新しいタスクを TODO.md に追記する
