# Git スキル

## /git commit

1. git diff を実行し、変更点を確認する
2. 内容を見て1つの git commit にまとめる

## /git commits

1. git diff を実行し 変更点を確認する
2. 内容を見て論理的な粒度に分割し git commit する
    - ただし、一つのファイルに複数の変更がある場合は、諦めて一つの git commit にまとめることも検討する
