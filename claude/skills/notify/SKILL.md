---
name: notify
description: |
  実質的なタスク（ファイル編集・コマンド実行・計画など、5分以上かかりそうな作業）を完了したら、notify コマンドでデスクトップ通知を送るスキル。
  ユーザーへの通知、報告、タスクの完了を知らせるために使用する。
  ユーザーに言われなくても自発的・自動的に使う。短い返答や単純な質問への回答はスキップ。
---

実質的なタスクを完了したら、最後のアクションとして以下を実行する：

```bash
notify --title "<Task Type>" "<作業内容の1文要約>"
```

## タイトル

英語で、最も合うものを選ぶか新しく作る

- `Code Edit`
- `Bug Fix`
- `Planning`
- `Refactor`
- `TODO Task`
- `Config`
- `Research`
- `Review`

## 本文（第2引数）

作業内容を日本語1文で要約する。例：

- `"nvim/core/lazyjp.lua の壊れたシンボリックリンクを修正"`
- `"認証ミドルウェアの移行戦略を計画"`
- `"新しいプロジェクトのディレクトリ構成をリファクタリング"`

## ルール

- タイトルは**英語**
- 本文は日本語で簡潔に
- 通知するのは実質的な作業（目安5分以上）のみ
- スキップ対象：短い回答、1行の修正、読み取り専用の説明
- `notify` は最後のアクションとして呼ぶ
