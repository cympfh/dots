# private スキル

`~/Dropbox/memo/private/` 以下にプライベートな情報をMarkdownで管理する。

## ディレクトリ構成

```
~/Dropbox/memo/private/
├── health/     # 健康、食事、運動、体調
├── work/       # 仕事、タスク、メモ
├── money/      # 家計、支出、収入
├── daily/      # 日常の記録、気づき
├── hobby/      # 趣味、読書、ゲーム、音楽
├── people/     # 人間関係、連絡先
├── tech/       # 技術メモ、ツール設定（個人的なもの）
└── home/       # 家の中に関するもの
```

## 動作フロー

### 記録モード（引数が疑問文でない場合）

1. `~/Dropbox/memo/private/` がなければ `mkdir -p` で作成し、`git init` する
2. 内容からカテゴリを判断（上記8カテゴリから選ぶ。新カテゴリが必要なら提案してから作成）
3. カテゴリ内の適切なファイルを選ぶ（なければ内容に合ったファイル名で自動作成）
4. 既存ファイルはWikipedia形式で管理：日付で追記するのではなく、トピックごとに内容を常に最新状態に上書き・マージする
5. ファイルを追加・更新したら、そのカテゴリの `INDEX.md` と `~/Dropbox/memo/private/INDEX.md` を更新する
6. `git add -A && git commit -m "<変更内容の一行要約>"` でコミットする
7. 書いたファイルパスと内容を簡潔に報告

## INDEX.md の仕様

- 全ディレクトリ（カテゴリ）は `INDEX.md` を持つ
- `~/Dropbox/memo/private/INDEX.md`：全カテゴリと各ファイルへのリンク一覧（トップレベル目次）
- `~/Dropbox/memo/private/<category>/INDEX.md`：そのカテゴリ内のファイル一覧と各ファイルの概要
- ファイルを追加・更新するたびに関係する INDEX.md を最新状態に保つ

### 質問モード（引数が疑問文「～？」の場合）

1. `~/Dropbox/memo/private/` 以下を検索
2. 関連する内容を見つけて回答
3. 該当ファイルのパスも提示

## ルール

- ファイル形式はMarkdown（`.md`）固定
- 確認なしで即書き込む
- カテゴリ判断に迷う場合は `daily/` をデフォルトに
